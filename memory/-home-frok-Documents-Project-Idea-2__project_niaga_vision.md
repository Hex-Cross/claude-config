**Niaga** ("commerce" in Indonesian) — a unified, mobile-first, offline-capable business management platform for Indonesian mid-market SMEs (20-200 employees, IDR 2.5B-50B revenue).

**Why:** The #1 pain in Indonesian business is fragmented tools that don't integrate. 500K-1M addressable businesses, only 5-8% SaaS penetration. Mid-market gap between basic tools (Jurnal/Accurate) and enterprise (SAP/NetSuite).

**How to apply:** All implementation decisions should optimize for: (1) genuine cross-module integration (sale -> inventory -> journal -> tax invoice in one flow), (2) mobile-first UX (80%+ users on Android), (3) Indonesian tax compliance as moat (Coretax, e-Faktur, PPh 21 TER), (4) WhatsApp as primary notification/approval channel, (5) offline capability for areas outside Java major cities.

Tech stack: Next.js 16 + Vercel + Neon Postgres + Clerk + AI SDK v6 + Chat SDK + Workflow DevKit. Bahasa Indonesia primary UI.
